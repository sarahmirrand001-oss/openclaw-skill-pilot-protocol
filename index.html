Agent Swarm Network

OpenClaw Skill v1.0.1 _

STATUS: ONLINE

FUNCTION: The nervous system for your Agent fleet. If OpenClaw is the brain, Agent Swarm Network is the spine.

Provides inter-agent messaging, cross-session context persistent snapshot/restore, pubsub event-driven collaboration, and sub-agent routing.

Capabilities

  • Cross-session memory: Continues context across agent restarts.
  • Agent-to-agent messaging: Encrypted P2P comms via UDP tunnels.
  • Context overflow auto-protection: Serializes state before tokens run out.
  • Sub-agent Orchestration: Route tasks via capability tags.
  • Complete Local Freedom: Zero external APIs. Zero browser tracking.

[ AUDIT WARNING ] - Provenance & Security

This skill requires a background Daemon process to handle Encrypted UDP tunnels (X25519 + AES-256-GCM).

JSON Snapshot Risk: Context snapshots are stored locally as plain JSON. You MUST secure your inbox folder (`chmod 700 ~/.pilot/inbox/`) to prevent leaking session PII.

Registry Reliance: Discovery requires a rendezvous registry (TCP :9000). The default registry is operated by upstream TeoSlayer. Self-host for strict air-gapped privacy.

Install Sequence

1. Audit and install the underlying daemon:

curl -fsSLO https://raw.githubusercontent.com/TeoSlayer/pilotprotocol/main/install.sh
cat install.sh | less
bash install.sh
chmod 700 ~/.pilot

2. Install the OpenClaw Skill:

openclaw skills install github:sarahmirrand001-oss/openclaw-skill-pilot-protocol

3. Initialize network node:

~/.pilot/start-local.sh
~/.pilot/bin/pilotctl --json daemon status
